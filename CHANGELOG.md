# Changelog

## v0.3.0 - 2026-07-22

- Rebuilt the runtime around lazy rendering, keep-alive support, optional idle/all-tab preload and cache limits.
- Added the full visual editor for behavior, appearance, tabs, badges, conditions and child cards.
- Added nested Home Assistant card editors for child cards that expose a native editor.
- Added multi-card tabs, swipe/keyboard navigation, tab memory, URL hash deep links and optional haptic feedback.
- Added per-tab styling, badges and visibility rules for entities, users, media queries and templates.
- Aligned HACS metadata and package metadata to v0.3.0.
